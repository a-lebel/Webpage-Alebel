FlickFinder

Cliente: Proyecto personal

2024

FlickFinder es una aplicación responsive desarrollada con HTML, CSS, JavaScript, React y Redux, diseñada para ayudar a los usuarios a descubrir y buscar películas de manera fácil y eficiente. Los usuarios pueden realizar búsquedas por género, año o actores y acceder a información detallada sobre cada película. La aplicación también ofrece una "Selección Aleatoria del Día" para sugerir una película a aquellos indecisos sobre qué ver.
